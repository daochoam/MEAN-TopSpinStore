# TopSpin Store
El proyecto describe una aplicación WEB e-commerce de productos deportivos de tenis de mesa.
## Tecnologías Empleadas
* HTML 5.
* CSS 3.
* OpenJDK 17.0.5 2022-10-18.
* ExpressJS >= 4.18.2.
* MondoDB >= 5.0.14.
### devDependencies
* body-parser => 1.20.1.
* cors => 2.8.5.
* mongoose => 6.9.0.
* nodemon => 2.0.20.
